Notification Manager System

Goal: Create a robust notification system that manages multiple alerts displayed on a screen (simulated, e.g., via console logs).

The Task: Create a NotificationManager object.

Requirements:

  • It must have a Set to store active notification IDs (ensuring uniqueness).
  • Implement an add(message, duration) method that creates a notification object and uses setTimeout to automatically trigger its removal after the specified duration.
  • Implement a clearAll() method that must use clearTimeout on every pending notification to prevent memory leaks before wiping the system state.

Active IDs: none
