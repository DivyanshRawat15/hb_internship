The Deep Merge & Clone Engine

Task: Write a function deepUpdate(originalObj, updates).

Requirements:
Use Object.entries() to loop through the properties of the updates object.
If a value in updates is another object, you must recursively merge it.
Use the Ternary Operator to check if a property exists before overwriting it.
Constraint: You cannot mutate the originalObj. You must return a brand new object using the Spread Operator (...) at every level to ensure a deep clone/update.
